# Figurinha Copa 2026

## Setup
1. Clone o repositório
2. `npm install`
3. Crie `.env.local` com: `FAL_KEY=sua_chave`
4. Coloque as imagens em `/public/`: figurinha-helena.png, figurinha-miguel.png, figurinha-arthur.png, body-base.jpg, loading-video.mp4
5. `npm run dev`

## Deploy no Vercel
- Adicione a variável de ambiente `FAL_KEY` no painel do Vercel
